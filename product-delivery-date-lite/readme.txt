Allow the customers to choose a delivery date on the frontend product details page for WooCommerce store owners.


== Description ==

This plugin will allow the customer to choose a delivery date on the frontend product details page. The customer can choose any delivery date that is after the current date. The plugin uses an inbuilt datepicker that comes with WordPress.

The plugin allows the site administrator to enable/disable the delivery date capture on the fron end product details page.

This plugin allows you to improve your customer service by delivering the product on the customer's specified date.

== Installation ==

1. Ensure you have latest version of WooCommerce plugin installed
2. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enable the Product Delivery Date calendar from the admin Products -> Add products page.
